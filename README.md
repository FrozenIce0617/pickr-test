# Pickr Test

Pickr Test Project

## Project Setup

### Project stack:

-   React Native
-   Redux
-   React Native Elements
-   Typescript

### Prerequisites

```
node v8^
npm
yarn
```

### Install and Run in Local

Note: Use `yarn` rather than `npm`

Install dependencies with

`yarn add`

For only `iOS`

`yarn pod:install`

Start App for `iOS`

`react-native run-ios`

Start App for `Android`

`react-native run-android`
